# trainingtask



## Системные требования

- iOS 11.0+
- Xcode 13.1
- Git
- Java 18

## Инструкция по установке


1. Копируем удаленный репозииторий
```
git clone https://git.qulix.com/AgeenkoEG/trainingtask.git
```
2. Заходим в папку проекта
```
cd trainingtask
```
3. Делаем билд 
```
./gradlew build xcodebuild archive package
```
